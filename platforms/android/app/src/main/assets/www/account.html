Help Info

You can update account information or view the billing invoice. You can review the system messages from the message icon on the top if there are messages. It will notify you if the Buy or Sell transaction signal for the stock has changed.

Features:

  • View My Stock List - display my current added stock list
  • Configuration - update email, password or change plan
  • Billing Invoice - show the past bill invoices
  • View Fund Mgr Subscription - show best fund manager portfolio

If you have a Mutual Fund Portfolio account, the system will select 4 best stocks automatically from the trading account based on the portfolio balance calculation. This will represent your investment portfolio in total amount.
