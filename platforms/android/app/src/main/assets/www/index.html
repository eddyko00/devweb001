Introduction

Now we have a great opportunity to use neural networks in trading strategy. This application provides stock trading predication based on the historical data from market data feed. It will apply the neural network AI filter to the trading signal to determine the best result. After the analysis is over, the system will continue to train new market data to adapt the market condition.


Features:

  • Using Neural Network AI Predication for best result
  • Provide RESTFUL API service for business use
  • Develop custom model AI filtering upon request
