Introduction

Now we have a great opportunity to use neural networks in trading strategy. This application provides stock trading signal predication based on the historical data from market data feed. The program combines both Neural Network AI and Technical Analysis to tactical choices on the timing of buying and selling.


What problem are we trying to solve?

  • Technical Analysis is an excellent on historical track records because they were created to fit old data. However, markets is keep changing, and this strategy will not work in the future.

Solution:
  • The application is using neural network AI to train the technical signal to correct the best price for buying and selling. This process will continue to learn the best trading signal every day to adapt the market change.

Features:

  • Using Neural Network AI Predication for buying and selling to achieve best result
  • Provide RESTFUL API service for business to create new business opportunities
  • Develop custom AI model for the user trading strategy upon request
