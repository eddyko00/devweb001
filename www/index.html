IISWEB System

Introduction

Now, we have a great opportunity to use Neural Networks AI in trading strategy. This Intelligent Investment System (IIS web) provides stock trading signal predication based on the historical data from market data feed. The system will combine both Neural Network AI and Technical Analysis to select the best signal for the buying and selling.


What problem are we trying to solve?

  • Technical Analysis is an excellent on historical track records because they were created to fit old data. However, market is keep changing and this strategy will not work in the future based on the static model.

Solution:
  • This application is using neural network AI to learn the best stock price for buying or selling. The Neural Network algorithm is to learn from the past trading signals and then update the new decision logic. This process will continue to learn the best trading signal every day to adapt the market changes.

Features:

  • Using Neural Network AI Predication for buying and selling to achieve best result
  • Provide RESTFUL API service for business to create new business opportunities
  • Develop custom AI model for the user trading strategy upon request
